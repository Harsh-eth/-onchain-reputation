# OnChain Reputation

a small on-chain reputation system built in solidity to experiment with state management and tier-based scoring

## What It Does

- users can register
- accounts can be activated or deactivated
- registered users can earn points
- points determine user tier
- public getters to check user data

## Why I Built This

i wanted to understand how to manage user state on-chain and design a simple tier logic system without overcomplicating the contract

this project helped me practice access control, mappings, events and basic security patterns

## Tech Used

- Solidity
- HTML / CSS
- JavaScript (ethers.js)
